# ARCADE
Repositório com dataset e códigos do Projeto ARCADE

## Conteúdo

- `data/` → Base de dados anonimizada em formato CSV
- `GPSK/` → Implementação do GPSK
- `RNA/` → Modelos de Redes Neurais Artificiais
- `CMA-ES/` → Otimização com CMA-ES

## Requisitos

- Python 3.9+
- Bibliotecas: pandas, numpy, datetime, matplotlib, sklearn, os, json, pathlib, typing, tensorflow, cma, re
- Adequar os nomes de diretórios e arquivos diretamente nos códigos
